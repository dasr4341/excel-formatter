Log sheet format

An all-in-one online Daily log editor to edit csv file to the required format. The simplest way to edit your time work log and do more in less time, so you can get down to job quickly.

  • Date has to be on mm/dd/yyyy format
  • The Hours has to be in hh:mm format
  • 1 empty row after each date
  • No duplicate dates
  • No repetition in date, if the above feature is the same date (no blank rows between items of the same date)
  • Please don’t have rows with items less then 1 hr (club the points in the same row if necessary)
